SortScout

SortScout is a smart product comparison website that helps users find the best deals by comparing the same product across multiple e-commerce platforms. Whether it's electronics, fashion, or daily essentials, SortScout makes online shopping easier by showing real-time prices, availability, and ratings—all in one place. Designed for speed, accuracy, and ease of use, it's your go-to tool for smarter shopping.
Created a JAR file for ease of distribution and deployed a static webpage using GitHub Pages to allow users to download the application seamlessly.
